# Zettelkasten

*Elke notitie is een atoom van kennis.*

## VIM commando's

- `:tag rust async` — vervang alle tags
- `:tag+ newtag` — voeg één tag toe
- `:tag- rust` — verwijder een tag
- `:retag` — herbereken tags uit #hash in tekst

## Navigatie

- `h j k l` — bewegen · `w/b` — woord
- `gg / G` — begin/einde
- `dd` — verwijder · `yy` — kopieer · `p` — plak
- `u` — undo · `Ctrl+r` — redo
